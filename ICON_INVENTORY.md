## üé® Branding Recommendations

For custom branded icons, prioritize these by visibility:

### Tier 1 (Hero icons - everywhere)
- `Truck` - Make this YOUR signature
- `MapPin` - Location identity
- `Star` - Quality signal
- `Heart` - Emotional connection

### Tier 2 (High-use feature icons)
- `Search`, `Filter` - Discovery flow
- `Clock`, `Utensils` - Food context
- `ChefHat` - Owner identity
- `Bell` - Engagement

### Tier 3 (System icons - can stay generic)
- `Check`, `X`, `Loader2` - Universal patterns
- `ChevronDown/Left/Right/Up` - Navigation
- `Settings`, `User` - Standard UI

## üîó Icon Source

All icons are from [Lucide React](https://lucide.dev/icons/).

To create custom icons, export SVGs from Lucide and edit them, or create your own SVGs with the same sizing (24x24 viewBox, 2px stroke width).
